== Description ==

This plugin catches PHP errors and exceptions and reports them to Ratchet.io.

== Installation ==

1. Create a (Ratchet.io)[https://ratchet.io/] account
2. Upload `ratchet.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's it!
